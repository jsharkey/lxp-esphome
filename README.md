This is a collection of ESPHome examples that connect to the RS485 Modbus connection of these popular solar hybrid inverters:

* EG4 18kPV-12LV
* EG4 6000XP
* EG4 12000XP
* LuxPower LXP-LB-EU 12K

Because there are hundreds of registers published by these devices, the bulk of the ESPHome sensor code is autogenerated using `lxp_auto.py`.  Each inverter model has a TSV file that summarizes the register configuration, including hiding irrelevant registers.

## Usage

The most user-friendly way to get started is using M5Stack ATOM and TAIL485 modules:

* [M5Stack ATOM](https://docs.m5stack.com/en/core/ATOM%20Lite)
* [M5Stack TAIL485](https://docs.m5stack.com/en/atom/tail485)

You can build and flash in the normal way:

> ```
> esphome run 12000xp.yaml
> ```

Then to connect to the inverter:

1. Connect 12V/GND from the TAIL485 to 12V/GND quick-connectors on the inverter.
1. Connect A/B from the TAIL485 to the inverter:
    * If your inverter has RS485 quick-connectors, you can connect directly to them.
    * Otherwise, find the 4-pin connector that routes out to the external HDMI dongle port, keeping track of the black and white wires.  Connect those black/white pins to A/B on the TAIL485.

## Advanced Usage

If you want to customize your inverter, edit the relevant TSV file to change things like `entity_category` and `internal`.  (Note that most registers are still read even when marked `internal` to help batch RS485 transactions.)

Then regenerate the ESPHome code using a command like this:

> ```
> python3 lxp_auto.py < 12000xp.tsv > 12000xp_auto.yaml
> ```

Note that the main file uses the `packages` feature to seamlessly merge in this autogenerated code.
